# World Clock for Aliens

A responsive, visually appealing world clock web application that displays the current time in multiple cities across various time zones. Users can manually set the time for the Indian Standard Time (IST) zone, and the app will update all clocks accordingly.

## Features

- Displays clocks for 36 major cities worldwide with clear analog designs.
- User input to set IST time manually, updating all clocks in real-time.
- Responsive grid layout adapts to different screen sizes.
- Smooth clock hand animations with hour, minute, and second hands.
- Reset button to revert to the actual current time.
- Scroll-based footer visibility and a smooth "Back to Top" button.
- Clean, modular codebase with separate HTML, CSS, and JavaScript files for easy maintenance.

## Usage

1. Open `https://namastevis.github.io/world-clocks/` in any modern web browser.
2. Use the time input to set IST time manually or reset to live time.
3. Scroll down to explore clocks and use the back-to-top button for easy navigation.

## Code Structure

- `index.html` — Markup and basic structure.
- `index.css` — Styling and responsive layout.
- `index.js` — Logic for rendering clocks, managing time zones, and user interactions.
